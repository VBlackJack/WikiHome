# WikiHome 🏠

"La documentation technique de la vie domestique. Concise. Structurée. Sans blabla."

## Description
WikiHome est une base de connaissances "No-Fluff" (pas de narration inutile) conçue pour la gestion efficace du quotidien.

**Structure type des articles :**
Prérequis > Étapes numérotées > Troubleshooting

## Structure des dossiers
* **Saisonnier** : Gestion des événements récurrents (Noël, vacances, saisons).
* **Bricolage** : Maintenance technique et réparations domestiques.
* **Admin** : Gestion administrative, impôts, documents.
* **Cuisine** : Procédures techniques, batch cooking, ratios (pas d'histoires culinaires).
