## Checklist Administrative Annuelle

**Date de mise a jour :** 2026-03-22
**Difficulte :** 1/5

### Calendrier des Echeances

| Mois | Echeance | Action |
| :--- | :--- | :--- |
| **Janvier** | Taxe fonciere (mensualisee) | Verifier le prelevement du 15 janvier |
| **Janvier** | Assurance habitation | Verifier la reconduction tacite, comparer les offres |
| **Fevrier** | Releve compteurs | Relever eau, gaz, electricite (photo avec date) |
| **Mars** | Entretien chaudiere | Planifier le RDV annuel **obligatoire** (attestation exigible par l'assurance) |

### Documents a Conserver (Durees Legales)

| Document | Duree minimale | Base legale |
| :--- | :--- | :--- |
| Bulletins de salaire | **Jusqu'a la retraite** (minimum 3 ans legalement, mais conserver indefiniment) | Code du travail |
| Avis d'imposition | **3 ans** (duree du droit de reprise du fisc) | Code general des impots |
| Releves bancaires | **5 ans** | Code de commerce |
| Quittances de loyer | **3 ans** apres depart du logement | Code civil |
| Factures (electricite, eau, telecom) | **2 ans** (5 ans pour les litiges) | Code de la consommation |
| Attestation d'assurance habitation | **Duree du contrat + 2 ans** | Code des assurances |
| Contrats de travail | **5 ans** apres fin du contrat | Code du travail |
| Acte de propriete | **A vie** (+ 30 ans apres la vente) | Code civil |
| Certificat de ramonage | **1 an** (exigible par l'assurance en cas de sinistre) | Reglement sanitaire departemental |

!!! danger "En cas de sinistre"
    L'assurance peut refuser l'indemnisation si le certificat d'entretien de la chaudiere ou le certificat de ramonage n'est pas a jour. Conserver ces documents soigneusement.

### Procedure : Archivage Annuel (Decembre)

1. Rassembler les documents papier de l'annee dans un dossier date.
2. Scanner les documents importants (factures, attestations, certificats).
3. Organiser par categorie : `Impots/`, `Assurances/`, `Logement/`, `Sante/`, `Travail/`.
4. Supprimer les documents dont la duree de conservation est depassee.
5. Verifier les abonnements actifs (streaming, telecom, assurances) — annuler les inutiles.

??? note "Depannage / FAQ"

    | Probleme | Cause | Solution |
    | :--- | :--- | :--- |
    | Document perdu | Pas de copie numerique | Demander un duplicata a l'organisme emetteur (delai variable) |
    | Echeance fiscale manquee | Oubli ou absence de prelevement | Payer avec majoration de 10%. Mettre en place le prelevement automatique |
    | Assurance refuse un sinistre | Certificat d'entretien manquant | Contester par courrier recommande. A l'avenir, archiver systematiquement |

### Notes

!!! tip "Dematerialisation"
    Activer les factures electroniques partout ou c'est possible. Configurer un dossier de telechargement automatique par categorie.
