## Entretien Saisonnier du Domicile

**Date de mise a jour :** 2026-03-22
**Difficulte :** 1/5

### Printemps (Mars-Avril)

#### Exterieur
- [ ] Nettoyer les gouttieres (feuilles mortes accumulees en automne/hiver)
- [ ] Inspecter la toiture (tuiles deplacees, mousse)
- [ ] Verifier les joints de fenetre (etancheite pour l'ete et la pluie)
- [ ] Premier passage de tondeuse (lame haute, 7-8 cm)
- [ ] Tailler les haies avant la periode de nidification (interdit de tailler d'**avril a aout** pour proteger les oiseaux nicheurs)

#### Interieur
- [ ] Depoussierer les bouches de VMC (aspirer les grilles)
- [ ] Nettoyer les filtres de la hotte aspirante (tremper dans eau chaude + degraissant)
- [ ] Retourner les matelas
- [ ] Verifier les detecteurs de fumee (test du bouton + pile)

#### Climatisation (si installee)
- [ ] Nettoyer les filtres des splits (eau tiede + savon doux)
- [ ] Verifier le bon fonctionnement avant les chaleurs
- [ ] Entretien professionnel obligatoire si puissance > 12 kW (tous les 2 ans)

---

### Ete (Juin-Juillet)

- [ ] Verifier l'arrosage automatique (programmation, fuites)
- [ ] Proteger les canalisations exterieures du soleil direct (risque de degradation PVC)
- [ ] Controler le niveau de l'adoucisseur d'eau (si installe)
- [ ] Nettoyer les volets roulants (rails, lames)

---

### Automne (Septembre-Octobre)

#### Chauffage
- [ ] **Purger les radiateurs** avant la premiere mise en chauffe (voir [guide purge](../bricolage/purger_radiateur.md))
- [ ] Verifier la pression du circuit (1 a 1,5 bar a froid)
- [ ] **Entretien chaudiere** : planifier le RDV annuel obligatoire (certificat a conserver pour l'assurance)
- [ ] Remplacer le filtre a air de la VMC double flux (si equipee)

#### Exterieur
- [ ] Nettoyer les gouttieres (feuilles mortes)
- [ ] Rentrer ou proteger le mobilier de jardin
- [ ] Vidanger et ranger le tuyau d'arrosage avant le gel
- [ ] Proteger les robinets exterieurs (manchon isolant)
- [ ] Dernier passage de tondeuse (lame basse, 5 cm)

#### Interieur
- [ ] Calfeutrer les fenetres si courants d'air detectes (joints adhesifs mousse ou silicone)
- [ ] Retourner les matelas (2e rotation annuelle)
- [ ] Verifier les detecteurs de fumee

---

### Hiver (Novembre-Decembre)

- [ ] Verifier que le mode **hors-gel** de la chaudiere fonctionne
- [ ] Deneiger les acces (obligation legale du proprietaire/locataire devant son domicile dans de nombreuses communes)
- [ ] Verifier les joints du garage (etancheite au froid)
- [ ] Controler la consommation energetique (releve compteur, comparaison avec l'annee precedente)
- [ ] Installer le [sapin de Noel](montage_sapin.md) (decembre)

---

### Recapitulatif Annuel

| Tache | Frequence | Mois |
| :--- | :--- | :--- |
| Purge radiateurs | 1x/an | Septembre-Octobre |
| Entretien chaudiere | 1x/an (obligatoire) | Septembre-Novembre |
| Ramonage cheminee | 1-2x/an (obligatoire) | Septembre + Fevrier |
| Nettoyage gouttieres | 2x/an | Avril + Novembre |
| Rotation matelas | 2x/an | Avril + Octobre |
| Test detecteurs fumee | 2x/an | Avril + Octobre |
| Filtres VMC | 1x/an | Octobre |
| Filtres climatisation | 1x/an | Avril |

### Notes

!!! danger "Obligations legales"
    L'entretien annuel de la chaudiere (gaz, fioul) et le ramonage sont **obligatoires**. L'assurance peut refuser l'indemnisation en cas de sinistre si les certificats ne sont pas a jour.
