## Citron : 15+ Usages Domestiques

**Date de mise a jour :** 2026-03-22
**Difficulte :** 1/5

Le citron (acide citrique naturel, pH ~2) est un detartrant, degraissant, desodorisant, antibacterien et blanchissant naturel. Plus doux que le vinaigre, il laisse une odeur agreable.

!!! info "Citron frais vs acide citrique en poudre"
    L'acide citrique en poudre (rayon menager) est plus concentre, plus economique et se conserve indefiniment. Le citron frais ajoute les huiles essentielles du zeste (parfum) et une action abrasive avec la pulpe. Les deux sont interchangeables pour le nettoyage.

---

### Detartrage

| Surface | Procedure | Temps |
| :--- | :--- | :--- |
| **Bouilloire** | 2 c.a.s. d'acide citrique (ou jus de 2 citrons) + eau, faire bouillir, laisser agir 30 min, rincer | 45 min |
| **Robinetterie** | Frotter avec un demi-citron directement sur les depots, laisser agir 15 min, rincer | 20 min |
| **Fond de casserole** (depot calcaire) | Jus de citron pur, faire bouillir 5 min, frotter | 10 min |
| **Machine a laver** | 8 c.a.s. d'acide citrique dans le tambour, cycle 60C a vide | 1 cycle |
| **Lave-vaisselle** | 2 c.a.s. d'acide citrique dans le fond de cuve, cycle a vide | 1 cycle |
| **Parois de douche** | Pulveriser jus de citron dilue 50/50 eau, laisser agir, essuyer | 20 min |

!!! tip "Acide citrique vs vinaigre pour le detartrage"
    L'acide citrique est **plus puissant** que le vinaigre blanc a concentration egale, sans l'odeur. Pour les depots de calcaire tenaces, privilegier l'acide citrique en poudre (2 c.a.s. par litre d'eau chaude).

---

### Nettoyage et Degraissage

| Surface | Procedure |
| :--- | :--- |
| **Planche a decouper en bois** | Frotter avec un demi-citron + gros sel (abrasif naturel). Laisser agir 10 min, rincer, secher debout |
| **Micro-ondes** | Bol d'eau + jus d'1 citron + les deux moities pressees, chauffer 3 min, laisser reposer 5 min porte fermee, essuyer |
| **Plan de travail** | Frotter avec un demi-citron, laisser agir 5 min, essuyer. Desinfecte et laisse une odeur fraiche |
| **Inox** | Frotter au citron, rincer, secher au chiffon microfibre dans le sens du brossage |
| **Cuivre et laiton** (ternis) | Demi-citron trempe dans du sel fin, frotter la surface, rincer immediatement, secher |

!!! warning "Surfaces incompatibles"
    Comme le vinaigre, le citron attaque le **marbre**, la **pierre naturelle** et l'**email ancien**. Ne jamais laisser du jus de citron secher sur ces surfaces.

---

### Desodorisant

| Usage | Procedure |
| :--- | :--- |
| **Refrigerateur** | Demi-citron pique de clous de girofle sur une coupelle, remplacer chaque semaine |
| **Poubelle** | Quelques rondelles de citron au fond du sac |
| **Micro-ondes** (odeurs persistantes) | Chauffer un bol d'eau citronnee 3 min, essuyer |
| **Canalisations** | Jeter les ecorces de citron usagees dans le siphon + eau bouillante |
| **Mains** (ail, oignon, poisson) | Frotter les mains avec un demi-citron, rincer a l'eau froide |

---

### Detachant et Blanchissant

| Tache | Procedure |
| :--- | :--- |
| **Rouille** (vetement) | Jus de citron + sel sur la tache, exposer au soleil 2 h, rincer, laver |
| **Rouille** (surface dure) | Demi-citron + sel fin, frotter, laisser agir 30 min, rincer |
| **Taches jaunes** (transpiration, col de chemise) | Jus de citron + bicarbonate en pate, appliquer sur la tache, laisser 1 h, laver |
| **Blanchir le linge** | Ajouter le jus d'1 citron dans le bac de lessive pour un cycle blanc |
| **Taches de the / cafe** (tasse) | Frotter avec un demi-citron + sel, rincer |
| **Taches sur le marbre** (exception) | Ne pas utiliser de citron. Utiliser du bicarbonate en pate + eau oxygenee |

---

### Cuisine et Conservation

| Usage | Procedure |
| :--- | :--- |
| **Empecher l'oxydation** (pommes, avocats, artichauts) | Quelques gouttes de jus de citron sur la chair coupee. L'acide ascorbique ralentit le brunissement enzymatique |
| **Nettoyer une rape** | Frotter avec un demi-citron apres usage (decolle les residus de fromage/zeste) |
| **Desodoriser le broyeur** (si installe) | Jeter des quartiers de citron congeles dans le broyeur |
| **Conserver les herbes fraiches** | Emballer dans un torchon humide + quelques gouttes de citron, refrigerer |

---

### Autres Usages

| Usage | Procedure |
| :--- | :--- |
| **Anti-fourmis** | Pulveriser du jus de citron sur les points d'entree et les passages |
| **Raviver les couleurs du bois** (meubles) | 1 volume jus de citron + 2 volumes huile d'olive, appliquer au chiffon, lustrer |
| **Nettoyer une grille de barbecue** | Demi-citron pique sur une fourchette, frotter la grille encore chaude |
| **Enlever les traces de calcaire sur les vitres** | Frotter avec un demi-citron, essuyer a la microfibre seche |

---

??? note "Depannage / FAQ"

    | Probleme | Cause | Solution |
    | :--- | :--- | :--- |
    | Le calcaire resiste au citron frais | Concentration en acide citrique trop faible | Utiliser de l'acide citrique en poudre (2 c.a.s./L) : bien plus concentre qu'un citron |
    | Traces blanches apres nettoyage au citron | Residus d'acide citrique cristallise | Rincer abondamment a l'eau claire |
    | Le cuivre noircit au lieu de briller | Citron laisse trop longtemps sans rincage | Rincer **immediatement** apres le frottage. Le citron oxyde le cuivre s'il seche dessus |
    | Irritation de la peau | Acide citrique concentre sur peau sensible ou petite coupure | Porter des gants pour les usages intensifs. Rincer les mains apres usage |

### Notes

!!! tip "Recycler les citrons presses"
    Ne jamais jeter un citron presse : le zeste sert a parfumer, la moitie pressee sert a frotter les surfaces (planche, robinet, inox). Dernier usage : jeter dans les canalisations + eau bouillante.

!!! info "Citron + sel = combo universel"
    Le sel fournit l'abrasion mecanique, le citron fournit l'attaque chimique (acide). Ensemble, ils nettoient le cuivre, la rouille, les planches a decouper et les taches organiques.
