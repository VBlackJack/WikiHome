## Vinaigre Blanc : 20+ Usages Domestiques

**Date de mise a jour :** 2026-03-22
**Difficulte :** 1/5

Le vinaigre blanc (ou vinaigre d'alcool, 8-14% d'acide acetique) est le produit menager le plus polyvalent. Acide, antibacterien, degraissant, detartrant, desodorisant.

!!! danger "Ne JAMAIS melanger avec de la Javel"
    Vinaigre + Javel = degagement de **chlore gazeux** (toxique, potentiellement mortel). Ne jamais utiliser les deux produits en meme temps ni sur la meme surface sans rincage complet entre les deux.

---

### Detartrage

| Surface | Procedure | Temps |
| :--- | :--- | :--- |
| **Bouilloire / cafetiere** | Remplir de vinaigre pur, faire bouillir, laisser agir 30 min, rincer 2 cycles a l'eau claire | 45 min |
| **Robinetterie** | Tremper un chiffon dans du vinaigre chaud, enrouler autour du robinet, laisser agir | 1-2 h |
| **Pommeau de douche** | Immerger dans un sac plastique rempli de vinaigre, attacher autour du col | 2-4 h |
| **Fond de casserole** (depot blanc) | Faire bouillir vinaigre pur 5 min dans la casserole, frotter | 10 min |
| **Parois de douche** (traces blanches) | Pulveriser vinaigre chaud, laisser agir, frotter a l'eponge | 30 min |
| **Machine a laver** | 1 L de vinaigre dans le tambour vide, cycle 60C | 1 cycle |
| **Lave-vaisselle** | 1 verre dans le fond de la cuve, cycle a vide | 1 cycle |
| **Fer a repasser** | Remplir le reservoir de vinaigre 50/50 eau, faire chauffer et vaporiser sur un chiffon | 15 min |

### Nettoyage

| Surface | Procedure |
| :--- | :--- |
| **Vitres et miroirs** | Pulveriser vinaigre pur ou dilue 50/50 eau, essuyer au papier journal ou microfibre |
| **Carrelage** | 1 verre de vinaigre dans un seau d'eau chaude. Serpilliere classique |
| **Plan de travail** (inox, carrelage, stratifie) | Pulveriser vinaigre dilue 50/50, essuyer |
| **Micro-ondes** | Bol d'eau + 3 c.a.s. vinaigre, chauffer 3 min, laisser reposer 5 min porte fermee, essuyer |
| **Four** | Pulveriser les parois, laisser agir 30 min, frotter. Pour les gros depots : pate bicarbonate d'abord, vinaigre ensuite |
| **Planche a decouper** | Frotter au vinaigre pur, laisser agir 10 min, rincer |
| **Poubelle** (odeurs) | Pulveriser vinaigre pur au fond apres nettoyage |

### Linge

| Usage | Procedure |
| :--- | :--- |
| **Adoucissant naturel** | 1 verre de vinaigre dans le bac adoucissant a chaque lavage. Pas d'odeur residuelle apres sechage |
| **Detachant** (taches fraiches) | Tamponner la tache avec du vinaigre pur avant lavage |
| **Raviver les couleurs** | Ajouter 1/2 verre dans le tambour pour les couleurs ternes |
| **Eliminer les odeurs** (transpiration) | Faire tremper le vetement 30 min dans eau + 1 verre de vinaigre avant lavage |
| **Desinfecter les torchons** | Trempage eau bouillante + vinaigre 30 min |

### Autres usages

| Usage | Procedure |
| :--- | :--- |
| **Desherbage** (joints de terrasse) | Pulveriser vinaigre pur a 14% par temps sec et ensoleille. Repeter si necessaire |
| **Anti-fourmis** | Pulveriser sur les passages et points d'entree |
| **Deboucher un siphon** (leger) | 3 c.a.s. bicarbonate dans le trou + 1 verre de vinaigre. Laisser mousser 30 min, rincer a l'eau bouillante |
| **Decoller du papier peint** | Pulveriser vinaigre chaud dilue 50/50, attendre 10 min, decoller |
| **Degivrer un pare-brise** | Pulveriser 2/3 vinaigre + 1/3 eau la veille au soir (prevention) |

---

!!! warning "Surfaces incompatibles"
    Ne **jamais** utiliser de vinaigre sur : **marbre**, **pierre naturelle** (calcaire, granit poli, travertin), **joints en silicone** (accelere la degradation), **aluminium brut** (oxydation). L'acide attaque ces materiaux de facon irreversible.

??? note "Depannage / FAQ"

    | Probleme | Cause | Solution |
    | :--- | :--- | :--- |
    | Le calcaire ne part pas | Vinaigre trop dilue ou depot trop ancien | Utiliser du vinaigre pur a 14%, chauffer avant application, prolonger le temps de contact |
    | Odeur persistante apres nettoyage | Surface poreuse ou vinaigre trop concentre | Rincer abondamment. L'odeur disparait en sechant |
    | Traces blanches sur le carrelage apres sechage | Vinaigre trop concentre, residus mineraux | Diluer davantage. Rincer a l'eau claire apres nettoyage |

### Notes

!!! tip "Vinaigre 8% vs 14%"
    Le vinaigre menager a 14% est 2x plus concentre que le vinaigre de cuisine (8%). Pour le detartrage intensif et le desherbage, utiliser le 14%. Pour le nettoyage courant et le linge, le 8% suffit.

!!! info "Pourquoi ca marche"
    L'acide acetique dissout le carbonate de calcium (calcaire), degrade les graisses par saponification, et cree un pH hostile aux bacteries. Efficace, non toxique, biodegradable.
